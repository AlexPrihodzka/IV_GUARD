INTERVALE.GUARD предоставляет комплексное решение по защите ваших приложений
от вмешательства такого-то такого-то
со своей стороны мы гарантируем полную безопасность или что-то в этом роде
придумать дальше лонг дескрипшн
INTERVALE.GUARD это:
Полная безопасность
Любой запрос API проходит предварительную проверку, не собираем данные пользователей
Легкая интеграция
UI - подключение библиотеки в одну строчку, BackEnd - получение оценки поведения пользователя перед выполнением бизнес логики
Уникальный алгоритм
Алгоритм валидности браузера уникален для каждого пользователя
Как это работает
1. При инициализации UI библиотеки IV.GUARD, клиент получает одноразовый токен, и уникальный обфусцированный скрипт от сервера IV.GUARD
2.Клиент выполняет скрипт в фоне, и отсылает результат выполнения в сервис IV.GUARD, сервис в свою очередь проверяет результат вычислений клиента, и привязывает результат к токену.
3. Скрипт всегда генерируется уникальный, результатом его выполнения является шифрованная строка, шифр меняется для каждого обращения к серверу. К тому же скрипт доступен для получения 1 раз, после чего он удаляется из файловой системы сервера IV.GUARD
4. При обращении к API, клиент должен вместе с параметрами отправить токен который сгенерировался на первом этапе
5. API перед выполнением своей бизнес логики, например отправки СМС, отсылает запрос на сервер IV.GUARD с уникальным токеном для получения SCORE пользователя
6. Если score равен 1, значит пользователь выполнил скрипт, и данному пользователю можно доверять
